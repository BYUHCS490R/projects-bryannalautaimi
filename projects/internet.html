Internet

A global network consisting of many different computer networks where resources can be shared.

Web

Resources and content accessed through the internet

How are the Internet and Web different?

The Internet is the actual network in which resources are accessed, and the Web is the resources itself that isaccessed through this network.

ASCII and Unicode

Encoding schemes, but ASCII is an older standard for encoding English text and Unicode is a modern standard that can encode characters from all writing systems.

What is UTF-8?

An HTML tag that enables the use of emojis and and all other characters in the Unicode Standard.

URL

A unique address given to every web page.

Domain Name

The easily readable part of a URL of a website.

HTTP

Set of rules browsers use to fetch and send resources throughout a network through the internet.

HTTPS

Like HTTP, a set of rules browsers use to fetch and send resources throughout a network through the internet, but a more secure version.

IP Address

A unique number given to every device that is connected to the internet to identify them.

DNS

Converts domain names into numerical IP addresses so computers can understand.

W3C Recommendations

A set of guidelines that is endorsed by the World Wide Web Consortium which shows that it is ready to be implemented worlwide.

XML, XHTML, HTML, HTML5

All markup languages, XML is used to communicate between different systems and applications, XHTML is another version of HTML that is more developed to bring HTML and XML together, HTML is the core markup language use to display information on web pages, and HTML5 is the latest version of HTML that supports new features and elements.
