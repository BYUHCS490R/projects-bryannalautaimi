CSS Selectors

Different Ways to Select an Item

  • Tag Name
  • Descendant Rule
  • ID
  • Class

An ID is referred to in the CSS file starting with a hashtag. They also cannot be given twice to different tags, an ID can only be used in one. A class however is referred to in the CSS file starting with a period and can be used in more than one tag.

A DIV tag cause line breaks automatically, thinking about it like a box. The SPAN tag allows us to format a specific part of text inside a tag, not causing line breaks.

The DIV tag is a formatting tag, only for formatting purposes. The HEADER, MAIN and FOOTER are tags that describe the content sections of the webpage.
