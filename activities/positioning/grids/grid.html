Describe the parent-child relationship in HTML and why it becomes essential when using flex and grid.

It is up to the parent to place where the child goes. A tag is a parent when theres other elements inside of it, and these elements is the child. The parent is kind of like a container and the child(ren) are the items inside. It is essential when using flex and grid because it makes the HTML much more organized and easy to apply CSS.
